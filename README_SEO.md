# VukaSport - Melhorias de SEO

## Resumo

Este diretório contém a página de download otimizada para SEO da aplicação VukaSport, juntamente com arquivos de configuração essenciais para melhorar a discoverabilidade nos motores de busca.

## Arquivos Incluídos

### Arquivos Principais

O arquivo `index-seo-optimized.html` é a versão otimizada da página de download com as seguintes melhorias:

- Meta tags aprimoradas com palavras-chave relevantes
- Dados estruturados em formato JSON-LD (Schema.org)
- Semântica HTML melhorada com tags semânticas
- Atributos ARIA para melhor acessibilidade
- Open Graph e Twitter Card tags para redes sociais

### Arquivos de Configuração

O arquivo `manifest.json` define a configuração de Progressive Web App (PWA) com nome, descrição, ícones e palavras-chave otimizadas para discoverabilidade.

O arquivo `robots.txt` fornece instruções aos rastreadores de motores de busca sobre quais páginas indexar e com que frequência.

O arquivo `sitemap.xml` lista todas as páginas importantes do site com prioridades e frequências de atualização.

### Documentação

O arquivo `SEO_OPTIMIZATION_GUIDE.md` contém um guia completo com estratégia de palavras-chave, otimizações técnicas, recomendações de conteúdo e métricas a acompanhar.

O arquivo `seo-plan.md` apresenta o plano estratégico inicial de SEO.

## Como Implementar

### Passo 1: Substituir o arquivo HTML

Fazer backup do arquivo original e usar a versão otimizada. O arquivo `index-seo-optimized.html` pode ser renomeado para `index.html` após testes.

### Passo 2: Adicionar arquivos de configuração

Os arquivos `manifest.json`, `robots.txt` e `sitemap.xml` devem estar no diretório raiz do servidor web.

### Passo 3: Verificar no Google Search Console

Aceder a https://search.google.com/search-console, adicionar a propriedade do site, submeter o sitemap e verificar erros de indexação.

## Principais Melhorias Implementadas

### Dados Estruturados

O Schema.org SoftwareApplication define a app como entidade de software com nome, descrição, categoria, sistemas operacionais e URLs de descarregamento. O Organization Schema estabelece a identidade corporativa. O BreadcrumbList Schema melhora a navegação estruturada.

### Meta Tags Otimizadas

A descrição foi aprimorada com palavras-chave relevantes. Open Graph tags foram adicionadas para redes sociais. Twitter Card tags melhoram a apresentação em redes sociais. Hreflang tags suportam múltiplas variantes de português.

### Semântica HTML

O uso de tags semânticas como `<main>`, `<header>`, `<section>`, `<article>` e `<footer>` melhora a estrutura. Atributos ARIA aumentam a acessibilidade. Atributos alt em imagens descrevem o conteúdo visual.

### Arquivos de Configuração

O manifest.json define a PWA com palavras-chave. O robots.txt controla o rastreamento. O sitemap.xml lista todas as páginas importantes.

## Palavras-Chave Alvo

As palavras-chave primárias incluem VukaSport, Resultados desportivos tempo real, Jogos de Mabote, App desporto Moçambique e Moçambola 2026.

As palavras-chave de cauda longa incluem Acompanhar resultados de Mabote em tempo real, App de futebol para Moçambique, Notificações de jogos de Mabote e Estatísticas desportivas Moçambique.

## Próximos Passos

A criação de conteúdo expandido é essencial, incluindo seção de FAQ, blog sobre desporto e testemunhos de utilizadores.

A otimização de performance deve incluir implementação de cache, uso de CDN para imagens e minificação de CSS/JS.

A estratégia de link building deve incluir submissão a diretórios de apps, contacto com bloggers de desporto e criação de comunicados de imprensa.

O monitoramento contínuo deve rastrear rankings, impressões, cliques, CTR, tempo no site e taxa de rejeição.

## Métricas a Acompanhar

O alvo para posições de ranking é Top 3 para palavras-chave principais. As impressões devem crescer mensalmente 10%. Os cliques devem crescer mensalmente 15%. O CTR deve ser superior a 3%. O tempo no site deve ser superior a 2 minutos. A taxa de rejeição deve ser inferior a 50%.

## Ferramentas Recomendadas

As ferramentas essenciais incluem Google Search Console, Google Analytics 4, Lighthouse, Screaming Frog e SEMrush ou Ahrefs.
